# 工作簿负责人以最新一日为准，禁止沿用过期归属

## 背景

07-08 研究「页面停留时长」时，按 **07-06/07-08** 工作簿写「#6 蓝猫主责、非又初线」。主人贴出 **07-12** 工作簿：该项已改为 **【又初】**（序号 #9 用户停留时间），同时 #4 结算·归因、#5 标签亦为又初。写周报仍写「蓝猫主责」= 未读最新群消息/工作簿。

## 坑 / 错误做法

- 用几天前工作簿断定归属，不核对**最新一日** 09:00 簿
- 群里工作簿点名后，周报/日报仍按旧负责人写
- 因「研究时非我主线」就把该项从周报主责里删掉

## 正确做法

1. 涉及「谁负责」→ 先查**最新**每日工作簿（群消息 / provenance / 热 context），日期以簿头为准
2. 簿内标【又初】= 又初主责，不论前几日是否标过别人
3. 写日报/周报：又初名下项必须进完成/计划/卡点；改派要写清「自 YYYY-MM-DD 起改派」
4. 归档断档时主动问主人或扫群，禁止臆测
