Proactively save any durable guidance, validated workflow preferences, and stable project context that is likely to matter in later sessions, so the user does not have to repeat prior instructions after reopening the computer.

**Why:** The user explicitly said to save what needs remembering and not make them repeat previously stated context when work resumes later.

**How to apply:** When a conversation produces reusable rules, confirmed validation strategies, or stable project decisions, store them in memory before the session ends. Avoid saving ephemeral intermediate task state, but preserve durable collaboration preferences and long-lived project context.
