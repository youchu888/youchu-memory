# 工作日历 · 中国大陆（又初团队）

> **本地参考** · `~/.dc-platform/memory/` · 不进 Git  
> 权威来源：[国务院办公厅节假日安排](https://www.gov.cn/) · 国办发明电〔2025〕7号（2026）

## 统计口径

| 概念 | 规则 |
|------|------|
| **自然周** | **周一 00:00 ~ 周日 24:00**（Asia/Shanghai），ISO 8601 周序号 `YYYY-Www` |
| **常规工作日** | **周一至周六**（6 天） |
| **常规休息** | **周日**（除非国务院通知调休上班） |
| **法定假日** | 按国务院当年度安排，**全天放假，不计工作日** |
| **调休上班** | 国务院指定的周末/假日补班日，**计工作日**，需写日报 |

## 周报 / work-log 怎么用

1. **周文件** `work-log/YYYY-Www.md` 覆盖**整个自然周**（周一～周日）的汇总与计划。
2. **日流水** `YYYY-MM-DD.md`：仅在**实际工作日**有交办任务时写；法定假日默认不写（有紧急交办除外）。
3. **周报统计「本周完成」**：合并该自然周内**所有工作日**的 work-log（含调休日，排除法定假日）。
4. **「本周计划」**：只排**剩余工作日**；遇法定假日跳过，遇调休补班则纳入。
5. **出勤/日报缺漏**：不对法定假日要求日报；周日仅调休上班日要求。

## 示例：2026-W25（06-15 周一 ~ 06-21 周日）

| 日期 | 星期 | 性质 |
|------|------|------|
| 06-15 | 一 | 工作日 |
| 06-16 | 二 | 工作日 |
| 06-17 | 三 | 工作日 |
| 06-18 | 四 | 工作日 |

> W25 有效工作日 **4 天**（15～18）；19～21 端午放假。06-22 起为 W26。

## 命令（Agent 取自然周边界）

```bash
# 本周 ISO 周序号
TZ=Asia/Shanghai date +%G-W%V

# 某 ISO 周的周一、周日（需 python3）
python3 -c "
from datetime import date, timedelta
import sys
y,w=map(int, sys.argv[1].split('-W'))
d=date.fromisocalendar(y,w,1)
print(d, d+timedelta(days=6))
" 2026-W25
```
